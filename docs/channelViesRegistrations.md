# [API](README.md).channelViesRegistrations - Flow Commerce API - Node SDK Documentation

Result of looking up a specific tax registration number. Indicates validity of a number for a specific country and (if valid) the associated person/company.

## Usage

```JavaScript
import client from '@flowio/node-sdk';

const channelViesRegistrations = client().channelViesRegistrations;
```
